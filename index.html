Max Blackhall
Graduate Software Engineer

This webpage serves as a portfolio of my past programming projects. They have a variety of themes, including: web development, machine learning, and computer graphics. There are a mixture of academic and personal projects represented here.

Please feel free to contact me if you would like to discuss any of these programs.

Mini-C

A compiler for 'Mini-C'. A small, C-like programming language. Implemented using C++ and LLVM.

Better Bills

A web application for helping a group of people to divide bills fairly. Implemented using HTML, CSS, JavaScript, and SQL operating on a bespoke database schema.

Multiclass Perceptron

A custom-built multiclass perceptron, being used to classify iris flowers. Uses the kernel trick to allow for non-linear decision boundaries. Implemented using Python with NumPy, and visualised using MatPlotLib.

PID Controller

An implementation of PID controller for a simulated turtlebot in the Gazebo environment. Includes a particle filter for localisation. Created with Python using RosPy.

Procedural Terrain Generator

Uses a custom implementation of a perlin noise generator to produce a heightmap, which is rendered as a three dimensional graphic. The user can move around the environment using the keyboard. Developed using C++ with OpenGL.

Tetris

An implementation of Tetris, the classic video game. Includes a pseudo-3d effect from multiple perspectives. Uses C++ with OpenGL.

Data Structures

An implementation of a variety of common data structures. Created using Java.

Dissertation

An 18,000 word report discussing how Empirical Modelling techniques and concepts could be applied to the field of Educational Technology. Submitted in 2021 as my final year project.

Interpreter for Arithmetic

An interpreter for a programming language that facilitates arithmetic operations. Also allows the writer to define functions. Written in Java using JavaCC.

CV Classifier

A neural network that has been set-up to classify CVs into four categories depending on the strength of the candidate. Implemented using Python with TensorFlow.
